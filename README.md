# Sentiment-Bert

> Sentiment analysis model based on bert.

## Requirements

### Python

- Python@3.6
- scikit-learn@0.20.3
- numpy@1.15.4
- torch@1.1.0
- pytorch-pretrained-bert@0.6.2

### Pretrained model

Please read `README.md` under `bert` directory before you run the model.

## Run

Just run `python main.py`.

Alter configurations in `config.py`.
